# Virtual-Lab

Виртуальная лабораторная работа основанная на "РАСЧЕТ И ИЗМЕРЕНИЕ ПАРАМЕТРОВ ЭЛЕКТРОМАГНИТНЫХ ВОЛНВ НАПРАВЛЯЮЩИХ СИСТЕМАХ" часть 2 под названием "ВОЛНЫ Н11 И Е01 В КРУГЛОМ ВОЛНОВОДЕ".
ВЛР позволяет студентам, не имеющим возможность посещать очные лаболаторные работы, выполнить работу у себя дома за рабочим компьютером.
